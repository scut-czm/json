# 范例说明

- demo1.cpp 测试xml文件加载
- demo2.cpp xml保存到本地文件
- demo3.cpp 增删改查
- demo4.cpp 增删改查初级版本
- demo5.cpp 增删改查高版本

# 编译方式

```
g++  tinyxml2.cpp demo1.cpp -o demo1 -std=c++11
g++  tinyxml2.cpp demo2.cpp -o demo2 -std=c++11
g++  tinyxml2.cpp demo3.cpp -o demo3 -std=c++11
g++  tinyxml2.cpp demo4.cpp -o demo4 -std=c++11
g++  tinyxml2.cpp demo5.cpp -o demo5 -std=c++11
```
